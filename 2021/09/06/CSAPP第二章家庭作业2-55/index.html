题目

代码

#include <stdio.h>

void show_types(char *p, int len) {
	for (int i = 0; i < len; i ++) {
		printf("%x ", *p);
		p++;
	}
int main() {
	unsigned int x = 0x12345678;
	show_types((char*) &x, 4);
	return 0;
}

运行程序，控制台打印：78 56 34 12，输出由小端到大端逐个输出， 说明这是个小端机器

结论

小端机器。
