修改CentOS7hosts文件解决GitHub超时问题

2021-10-30 21:40:24#CentOS7

1.获取Github相关网站的ip

访问https://www.ipaddress.com，找到页面中下方的“IP Address Tools - Quick Links”，分别输入github.global.ssl.fastly.net

github.com

查询ip地址。

2. 修改CentOS 7 hosts文件

sudo vi /etc/hosts

添加如下两行

199.232.69.194 github.global.ssl.fastly.net
140.82.112.4 github.com

3. 刷新网络设置

/etc/init.d/network restart
