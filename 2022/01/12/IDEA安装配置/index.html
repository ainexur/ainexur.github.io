IDEA安装配置

1. 下载安装

  1. 官网下载：有Ultimate和Community两个版本选择，由于用于SpringBoot开发，这里选Community就足够（Ultimate版本需要购买激活）。
  2. 安装
  3. 启动

2. 配置

多行显示tabs：File | Settings | Editor | General | Editor Tabs

3. 快捷键

官方文档

单词操作

跳到单词前

Ctrl+←

跳到单词后

Ctrl+→

选中前面的单词

Ctrl+Shift+←

选中后面的单词

Ctrl+Shift+→

收起或展开方法

当前方法展开

Ctrl++

当前方法折叠

Ctrl+-

类方法全部展开

Ctrl+Shift++

类方法全部折叠

Ctrl+Shift+-

自动创建赋值对象

ctrl+alt+v

搜索

  • 双击Shift： Search Everywhere
  • Ctrl+Shift+A： Find Action
  • Alt+Enter： Show Context Actions
  • F2：Navigate between code issues
  • Shift+F2：Jump to the next or previous highlighted error.

查看类继承关系

Ctrl + H

查看当前类的所有方法

Alt+7
