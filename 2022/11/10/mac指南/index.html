mac指南

2022-11-10 21:19:31#mac

快捷键

Ctrl+A	#到行首（达到Home键的效果）
Ctrl+E	#到行尾（达到End键的效果）
Ctrl+N	#到下一行
Ctrl+P	#到上一行
Ctrl+K	#从光标处开始删除，直到行尾
fn +左方向键是HOME
fn键+右方向键是END
fn+上方向键是page up
fn+下方向键是page down

环境配置

#cnpm
sudo npm install cnpm -g --registry=https://registry.npmmirror.com

Git代理

# 设置如下
git config --global http.proxy 'socks5://127.0.0.1:1080'
git config --global https.proxy 'socks5://127.0.0.1:1080'

# 取消如下
git unset http_proxy
git unset https_proxy

VirtualBox

# 任意来源安装
sudo spctl --master-disable
# brew安装，amd黑果只支持7.0以下的版本，7.0以上使用了apple 虚拟框架，amd无法开启
brew install virtualbox@6.1.8 --cask
