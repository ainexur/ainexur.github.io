nginx-too-many-close-wait复现

2023-03-26 13:29:14#nginx#bug

惯例先贴地址：https://github.com/tonyo/nginx-too-many-close-wait

环境

WSL

  • Ubuntu 22.04.1 LTS on Windows 10 x86_64

  • Docker version 20.10.21

  • Python 3.10.6 [GCC 11.3.0] on linux

要求

  1. docker
  2. docker-compose
  3. python3

Docker重现步骤（未能重现）

1.拉代码

git clone https://github.com/ainexur/nginx-too-many-close-wait.git
cd nginx-too-many-close-wait

2.启动

# 配置linux内核最大连接数
sudo sysctl -w net.core.somaxconn=8
# 第一个窗口
docker-compose up
# 第二个窗口
./run_load_tester.sh
# 第三个窗口
docker-compose exec nginx watch 'netstat -atunpl | grep CLOSE_WAIT'
